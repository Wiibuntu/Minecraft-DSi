Place your background music .mp3 files here and rebuild.

Build behavior:
- music/*.mp3 -> converted to nitrofiles/music/*.wav at 11025 Hz, mono, unsigned 8-bit PCM
- music/*.wav -> converted/copied into nitrofiles/music/*.wav
- if music/ is empty, the build reuses existing nitrofiles/music/*.wav when present

The current uploaded source does not include any actual music tracks yet.
